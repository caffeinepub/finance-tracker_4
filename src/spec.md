# Specification

## Summary
**Goal:** Create a finance record-keeping application to track income and expenses with a clean, professional interface.

**Planned changes:**
- Implement backend data model to store financial records (date, description, amount, category, unique ID)
- Add backend functions to create, retrieve, and delete financial records
- Build frontend form to input new records with date picker, description, amount, and category selector
- Display all records in a table with delete functionality
- Calculate and show total income, total expenses, and net balance
- Design interface using warm earth tones (terracotta, sage green, cream) with card-based layouts

**User-visible outcome:** Users can add, view, and delete financial transactions, see them organized in a table, and track their total income, expenses, and net balance through a professional, visually cohesive interface.
